# .agents

Эта папка добавлена в соответствии с условиями челленджа и фиксирует, как использовались ИИ-инструменты во время работы над решением.

В работе использовались:

- Claude Code
- Codex

## Цель использования ИИ

ИИ использовался не для генерации “готового проекта целиком”, а как инженерный ассистент для:

- диагностики auth-flow и реального поведения backend на базе Ory Kratos
- поиска и диагностики ошибок auth-flow
- ускорения рефакторинга frontend-структуры
- подготовки и уточнения README
- генерации и доработки тестовых кейсов

## Что оставалось под ручным контролем

- выбор архитектурных границ
- проверка реального поведения backend
- принятие trade-offs
- верификация пользовательских сценариев
- финальная правка текста README и проектных assumptions

## Принцип работы

1. Сначала анализировался текущий код и реальный контракт backend.
2. Затем воспроизводилась ошибка на живом локальном окружении.
3. После этого вносился минимально достаточный фикс.
4. Изменения подтверждались тестами и ручной проверкой сценариев.

Подробности по основным шагам и типам задач вынесены в `worklog.md`.
